# html-to-pdf
turns a local html file into a pdf with headless browsers.<br/>

# installation
step 1: download google chrome if you haven't already<br/>
step 2: download .zip<br/>
step 2: extract<br/>
step 4: run cd {folder directory} in terminal for Linux users (or if you are a psychopath and don't use Linux, either  switch to Linux, or change directories how ever you do it in your OS).<br/>
step 5: run pip install .<br/>

# usage
html-to-python({FULL path of input file}, {FULL path of output file (will create path if doesn't already exist')})
